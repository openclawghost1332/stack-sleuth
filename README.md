# Stack Sleuth

Stack Sleuth is a static debugging helper that turns raw JavaScript, Python, and Ruby stack traces into a likely culprit frame, a short diagnosis, and a practical next-step checklist.

## Local development

1. Open `index.html` in a browser, or serve the folder with any static file server.
2. Run `npm test` to execute the Node test suite.
